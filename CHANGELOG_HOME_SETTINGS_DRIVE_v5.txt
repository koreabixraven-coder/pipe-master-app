배관기능장 앱 v5 수정 기록

1. 홈 화면에서 톱니바퀴 설정 버튼이 보이지 않는 문제 보완.
2. 홈 화면에 ⚙ 설정 바로가기 버튼 추가.
3. 홈 화면에 🎧 이동 중 청취 모드 바로가기 버튼 추가.
4. 설정 바로가기를 누르면 문제 풀기 화면으로 이동한 뒤 설정 패널을 자동으로 열도록 구성.
5. 이동 중 청취 모드를 누르면 문제 풀기 화면으로 이동한 뒤 이동 중 청취 모드를 자동 시작하도록 구성.
6. 기존 문제 데이터 1600문제 구조와 이론 데이터는 유지.
7. sw.js 캐시명을 v5로 변경하여 모바일 구버전 캐시 잔류를 줄임.
